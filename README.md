# V-Tool Suite

Professional 2FA Generator and UID Lookup utility with a clean, minimalist design.

## Features
- **2FA Authenticator**: Generate 2FA codes from secret keys locally.
- **UID Lookup**: Find Facebook UIDs from profile URLs.
- **History**: Keep track of your recent lookups.
- **Dark/Light Mode**: Switch between themes for your preference.

## Deployment to GitHub & Vercel

### 1. Export to GitHub
1. Open the **Settings** (⚙️ gear icon) in the top-right corner of AI Studio.
2. Select **Export to GitHub**.
3. Follow the prompts to connect your GitHub account and create a new repository.

### 2. Host on Vercel
1. Go to [Vercel](https://vercel.com/) and log in with your GitHub account.
2. Click **Add New** -> **Project**.
3. Import the repository you just created.
4. Vercel will automatically detect the settings.
5. Click **Deploy**.

### 3. Environment Variables
If you have any API keys or secrets, make sure to add them in the Vercel project settings under **Environment Variables**.

## Local Development
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
